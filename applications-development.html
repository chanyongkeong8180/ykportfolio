Bus Ticketing System

This is a bus ticketing system website developed using Node and Express JS. MySQL is also used for database integration with the website.

It allows users to book or cancel bus tickets and view upcoming bus tickets that the user has booked. It also allows users to change the start and end locations, as well as the date and time of the bus ticket.

However, not only prevents users from cancelling their bus tickets on the day of the bus ticket but also prevents users from booking or changing the bus ticket to a past or present date, as well as having the same location for the start and end locations of the bus ticket.

Favourite List

This is a favourite list mobile application developed using React Native. It allows users to add items to their favourite list in the fruits or vegetables section.

It allows users to remove items from their favourite list in the fruits or vegetables section. However, an alert message will be displayed to confirm with the user before the item is removed from their favourite list in the fruits or vegetables section.

It allows users to change the name and description of the item from the fruits or vegetables section but prevents users from changing the item type. However, it does not prevent users from adding the same item and adding or changing the item with empty values.

Shopping List

This is a shopping list mobile application developed using React Native. It allows users to add items to their shopping list to calculate the total price spent for one or more items. However, it does not prevent users from adding the same item.

It allows users to remove a particular item or all items from their shopping list. However, an alert message will be displayed to confirm with the user before a particular item or all items are removed from their shopping list.

It allows users to change the name, quantity and price of the item. However, it prevents users from adding or changing items without an item name or with a quantity and price of zero value by displaying an alert message.
