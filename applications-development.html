Bus Ticketing System

This is a bus ticketing system website developed using Node and Express JS. MySQL is also used to integrate the database with the application.

It allows users to book bus tickets and view all upcoming bus tickets which they have booked. Not only it allows users to change the date and time but also the starting and ending location of the bus ticket or cancel the bus ticket.

However, not only it prevents users from cancelling their bus tickets on the day of the bus ticket but also prevent users from booking or changing the bus ticket to a past or present date as well as having the same location for the starting and ending location of the bus ticket.

Colour Quiz

This is a colour quiz cross platform mobile application developed using React Native. It consists of four questions and users have to identify the colour depending on the question asked.

A toast message will be displayed if users do not answer all the questions. However, an alert message will be displayed to show their quiz scores after they have answered all the questions.

There will be different alert messages displayed depending on their quiz score. If users have scored one or below, the alert message will ask them to try again. However, if users have scored two or above, the alert message will congratulate them.

Favourite List

This is a favourite list cross platform mobile application developed using React Native. It consists of two sections which are fruits and vegetables.

Each section consists of two items and each item consists of an item name and image belonging to the item.

An alert message will be displayed to show the health benefits of the fruit or vegetable when users tap on one of the fruit or vegetable images.
