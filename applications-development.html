Bus Ticketing System

This is a bus ticketing system website developed using Node and Express JS. MySQL is also used for database integration with the website.

It allows users to book or cancel bus tickets and view upcoming bus tickets that the user has booked. It also allows users to change the start and end location as well as the date and time of the bus ticket.

However, not only it prevents users from cancelling their bus tickets on the day of the bus ticket but also prevents users from booking or changing the bus ticket to a past or present date as well as having the same location for the start and end location of the bus ticket.

Favourite List

This is a favourite list mobile application developed using React Native. It allows users to add items to their favourite list and it will be added to the bottom of the section of the favourite list based on the item type selected by the user.

It allows users to remove a specific item from their favourite list and it will be removed from the section based on the specific item. However, before the specific item is removed from the favourite list, an alert message will be displayed to confirm if the user wants to remove the specific item.

It allows users to change the item name and description but prevents users from changing the item type. However, it does not prevent users from adding the same item.

Shopping List

This is a shopping list mobile application developed using React Native. It allows users to add items to their shopping list and it will be added to the bottom of the shopping list. However, it prevents users from adding items which contain fields with empty or zero values.

It allows users to remove a specific item from their shopping list and it will be removed from the shopping list. It also allows users to remove all items from their shopping list and all of them will be removed from the shopping list. However, before a specific item or all items are removed, an alert message will be displayed to confirm if the user wants to remove the item.

It allows users to change the item name, quantity and price but prevents users from changing the item which contains fields with empty or zero values. However, it does not prevent users from adding the same item.
