Bus Ticketing System

This is a bus ticket system website application developed using Node and Express JS as well as MySQL for the database.

It allows users to book bus tickets and view all upcoming bus tickets that they have booked. It also allows users to change the date and time as well as the starting and ending location of the bus ticket or cancel the bus ticket.

However, it prevents users from cancelling their bus tickets on the day of the bus ticket. It also prevents users from booking or changing the bus ticket to a past or present date as well as having the same location for the starting and ending location of the bus ticket.
