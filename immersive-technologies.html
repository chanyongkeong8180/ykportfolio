Overview of the Game

Escape From Library is an escape room game for users to escape the library by completing four puzzles. It aims to help users who are from primary school to improve their understanding on what they have learnt and their general knowledge.

Unique Selling Points

Escape From Library uses puzzles in the form of trivia and riddles for users to answer which requires them to know the four basic operators in mathematics, general knowledge and memory skills.

Game Loop

User will be spawned in the library facing a door which requires a password and will have to use the thumb stick of the left controller to face the librarian behind them.

User will have to complete the first three puzzles successfully before the librarian gives the clue for the password to unlock the door in the fourth and final puzzle to escape the library.

Puzzle Without Visual Effects Puzzle Description With Visual Effects Visual Effects Description
Puzzle 1: Math Question The librarian will provide a math question for the user to answer
with four options and one of them is the correct answer.
The user will need to answer the question to solve the puzzle
by triggering the correct option using the left or right controller
before proceeding to the second puzzle. 		Post-Processing effect is chosen to enhance the visual effects of the escape room
by having a bright and colourful visual style. When performing the visual style,
contrast and saturation were increase to 50 each and tonemapping is set to neutral.

Puzzle 2: Missing Words When the user triggers the bookshelf using the left or right controller,
an opened book will spawn with some blanks and helping words.
The user will need to use the left or right controller
to trigger those helping words to answer all questions correctly
from top to bottom before proceeding to the third puzzle.
Puzzle 3: Something odd User will have to use the thumb stick to face the third puzzle
from the second puzzle. However, a chat bubble appears above the desk
that something is odd. User will have to identify which item in the puzzle
is odd by triggering it using the left or right controller
before proceeding to the last puzzle.
Puzzle 4: Password Guessing The librarian will provide a clue for the password to unlock the door.
The user needs to enter the password by triggering the buttons on the door
using the left or right controller. When the user triggers the correct password,
the door will be opened for the user to escape the library.

Gameplay Scope

The escape room should take at least about half to an hour with solo player being the most recommended. The objective of the escape room is to finish all four puzzles successfully to escape the library.
